Gestión de Contenidos Audiovisuales  
**Programación Orientada a Objetos (POO) – Unidad 2**

Este proyecto implementa un sistema orientado a objetos para gestionar diferentes tipos de contenidos audiovisuales: películas, series, documentales, videos de YouTube y cortometrajes.  

El sistema permite representar:

- Películas  
- Series de TV  
- Documentales  
- Videos de YouTube  
- Cortometrajes  

**Propósito del sistema**

El objetivo principal es **modelar un entorno realista de contenidos audiovisuales**, permitiendo:

- Crear y organizar distintos tipos de contenido  
- Asociar actores, temporadas o investigadores según el tipo de contenido  
- Mostrar información detallada de cada elemento  
- Demostrar el uso correcto de jerarquías y relaciones entre clases
